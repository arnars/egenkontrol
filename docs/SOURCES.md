En post som "ikke gennemgået" betyder, at linket er registreret, men indholdet endnu ikke er fagligt analyseret til projektet.

## Kilder, der skal tilføjes

- [ ] Relevant risikoanalyseskema
- [ ] Eksempel på egenkontrolprogram
- [ ] Relevante branchekoder
- [ ] Materiale om sporbarhed og tilbagetrækning
- [ ] Materiale om allergener
- [ ] Materiale om temperaturstyring
- [ ] Virksomhedens egen, godkendte risikoanalyse
- [ ] Virksomhedens versionerede procedurer

## Skabelon for en kilde

```md
### Titel

- Organisation:
- URL eller dokumentreference:
- Emne:
- Version/udgivelsesdato:
- Dato for seneste gennemgang:
- Gennemgået af:
- Påvirker dele af systemet:
- Faglig fortolkning og begrænsninger:
- Beslutninger/ADR'er eller domænepunkter baseret på kilden:
```

Gem ikke ophavsretligt materiale i repositoryet uden tilladelse. Registrér i stedet præcis reference, version og en kort projektorienteret vurdering.
